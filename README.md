# Solana Mint Bot

## Setup
```sh
yarn install
```

## Launch my NFT

### Running locally.

```sh
cd launch-my-nft

# Tweak settings/tasks
vim config.json

# Build package
yarn build

# Install locally and run
# The package will be installed at ~/.yarn/bin/lmn-mint (or `which lmn-mint`).
yarn local
```
